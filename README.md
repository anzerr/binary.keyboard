### `Info`
These are the source files use to build this small little binary keyboard. The idea is you type out the binary code for the char and press enter.

- 1 arduino pro micro
- 3 mx cherry switches
- 1 m3 threaded insert
- 1 m3 screw
- some pla
- micro usb cable
